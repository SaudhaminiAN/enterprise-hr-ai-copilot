# Enterprise HR AI Copilot

An enterprise-grade HR AI Copilot built using:

- LangGraph
- LangChain
- ChromaDB
- FastAPI
- Streamlit
- Python 3.11

## Features

- Policy Question Answering
- Multi-Agent Architecture
- HR Email Generation
- Compliance Checking
- Document Summarization
- Retrieval-Augmented Generation (RAG)
